# BoberOS

ОС для бобров. Прямо в браузере. На полном серьёзе.

Это шутейная, но вполне рабочая операционка, которая живёт у тебя во вкладке. С рабочим столом, окнами, меню Пуск, установщиком, магазином приложений, играми, лицензированием и своей экономикой. Всё сохраняется в localStorage — между заходами ничего не теряется (если сам сохранил).

Работает на телефоне и на компе.

**Играть:** https://vanya8scripts.github.io/BoberOS/

---

## Что внутри

При первом запуске тебя встречает BIOS (да, как у настоящих компов — с проверкой памяти «веток» и поиском устройств). Потом — выбор языка, имя пользователя, аватарка (можно загрузить своё фото), редакция (Pro / Home / Max VK EDITION RUSSIA — чисто для красоты) и активация ключом.

Ключ: `VANYA8-BOBEROS`. Если не вводить — потом можно активировать за 500 монет в настройках.

Дальше — сплэш, загрузка, рабочий стол. Можно создавать несколько пользователей, у каждого своё сохранение.

### Приложения (45+)

BPaint (рисовалка), BoberStore (магазин), Мой бобёр (файлы, и если удалить папку BoberOS — будет синий экран смерти), Блокнот, Калькулятор, BoberChat (поболтать с бобром), BoberBrowser (с фейковыми сайтами), Терминал (с работающими командами), Часы (с таймером и секундомером), BoberTunes (плеер), Редактор Бобра (как regedit), Галерея, Диспетчер задач.

Из магазина качаются: 360 антивирус, Спим (магазин игр), BoberCoin (майнинг кликами), почта, погода, карты, BoberTube, офис, инженерный калькулятор, VPN, облако, переводчик (русский ↔ английский ↔ китайский ↔ бобрий ↔ барсучий), мониторинг, календарь, редактор кода. Ещё пачка «скоро» — BoberBank, BoberTaxi, BoberFood и т.д.

### Игры (24)

В Спим лежит всё подряд:

- **3D:** Бобр3D — шутер с рейкастингом, бегаешь по лабиринту и стреляешь барсуков
- **Паркур:** БоброПаркур — wall-jump, рывок, двойной прыжок, шипы, чекпоинты
- **Стрелялки:** КиберБобер (платная, неон), Бобр: Штурм (run-and-gun с боссом), КС2 (комп не тянет)
- **Аркады:** Флэппи Бобёр, Змейка, БоброБег (dino runner), Арканоид, Реакция Бобра, Пинг-Понг
- **Головоломки:** 2048, Тетрис, Сапёр, Судоку, Сокобан, Пятнашки
- **Слова:** Виселица, БобрВорд (wordle)
- **Логика:** Крестики-нолики, Четыре в ряд
- **Прочее:** Память, Симон, Камень-Ножницы-Бумага

### Прочее

- Тёмная тема, звуковые эффекты (Web Audio), своя аватарка из картинки
- Панель задач как в Windows 11: раскладка, сеть, громкость, тема, батарея, уведомления
- Экономия: кликаешь монеты в BoberCoin → выводишь на BoberSoft (для активации) или на Спим (для игр)
- КиберБобер стоит 3500 — надо накликать и вывести на Спим
- Система сохранений: установленные приложения не сохраняются сами, жми «Сохранить». При выходе спросит.
- BSOD, если удалить системную папку. С обратным отсчётом и бобром.

## Запустить локально

```bash
bun install
bun run dev
```

Открой `http://localhost:3000`.

Сборка статическая (для GitHub Pages):

```bash
bun run build
```

Результат в папке `out/`.

---

## Деплой на GitHub Pages

В репо уже есть `.github/workflows/deploy.yml` — при пуше в `main` всё само собирается и выкладывается. В настройках репо (Settings → Pages → Source) выбери **GitHub Actions**.

`basePath` в `next.config.ts` уже настроен на `/BoberOS` (только для продакшена, в dev работает на корне).

## Стек
