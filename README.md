# 🐹 BoberOS

**Шуточная операционная система для настоящих грызунов** — построена на Next.js 16, TypeScript, Tailwind CSS и Canvas.

BoberOS — это полноценная десктопная ОС прямо в браузере: с рабочим столом, окнами, меню Пуск, установщиком приложений, играми, лицензированием и экономикой монет. Всё работает end-to-end и сохраняется в localStorage.

🌐 **Играть онлайн:** https://vanya8scripts.github.io/BoberOS/

📦 **Исходники:** https://github.com/vanya8scripts/BoberOS

## ✨ Возможности

### 🖥️ Полноценная оболочка
- **Установщик (OOBE)** — выбор языка (🇷🇺 Русский / 🇬🇧 English), имя пользователя, аватар (эмодзи или **загрузка своего фото**), редакция (Pro / Home / Max VK EDITION RUSSIA), активация ключом `VANYA8-BOBEROS`
- **Splash-экран** с плавным fade и кнопкой продолжения
- **Загрузочный экран** с прогрессом и бобро-шутками
- **Рабочий стол** с обоями, иконками (wrapping), выделением прямоугольником
- **Окна** — перетаскиваемые, масштабируемые, сворачиваемые, с z-index
- **Меню Пуск** с поиском
- **Панель задач** в стиле Windows 11: кнопки приложений, раскладка (RU/EN), сеть, громкость, тёмная тема, батарея, часы + уведомления
- **Quick Settings** панель (как в Windows 11)
- **BSOD** при удалении системной папки `C:\BoberOS`

### 🎨 Темы и кастомизация
- **Тёмная / светлая тема**
- **5 обоев** (3 градиента + 2 сгенерированные изображения)
- **Загрузка своих обоев** — положите `wallpaper1.png` и `wallpaper2.png` в `public/`
- **Кастомный аватар** из любой картинки
- **Звуковые эффекты** (Web Audio API) — клики, открытие окон, монеты, победа/поражение

### 💰 Экономика
- **BoberCoin** — кликер: 1 клик = 100 монет
- **Вывод на BoberSoft** — для активации OS (500) и покупок в BoberStore
- **Вывод на Спим** — для покупки игр (КиберБобер = 3500)
- Полный цикл: клик → вывод → активация → покупка игр

### 🔑 Лицензирование
- Активация **ключом** `VANYA8-BOBEROS` (бесплатно при установке)
- Или **за 500 BoberSoft** в Настройках
- BoberStore **заблокирован** до активации + водяной знак на рабочем столе

### 💾 Система сохранений
- Установленные приложения **не сохраняются автоматически**
- Кнопка **"Сохранить"** в панели задач (с индикатором `*`)
- При выходе — предупреждение **"Хотите выйти? Сохранить прогресс?"**

### 📦 45+ приложений

**Системные:** BPaint, BoberStore, Мой бобёр (файловый менеджер), Блокнот, Настройки, Калькулятор, BoberChat (умный собеседник-бобёр), BoberBrowser, Терминал, Часы, BoberTunes (плеер), Редактор Бобра (registry), Галерея, Диспетчер задач

**Из BoberStore:** 360 Антивирус, Спим, BoberCoin, BoberMail, BoberWeather, BoberMaps, BoberTube, BoberOffice, BoberCalc Pro, BoberVPN, BoberCloud, BoberTranslate (русский ↔ английский ↔ **китайский** ↔ бобрий ↔ барсучий), BoberMonitor, BoberCalendar, BoberStudio (редактор кода)

**Скоро в магазине:** BoberBank, BoberShop, BoberTaxi, BoberFood, BoberFit, BoberPay, BoberSocial...

### 🎮 21 игра в Спим
- **Аркады:** Флэппи Бобёр, Змейка, Реакция Бобра, БоброБег (Dino Runner), Арканоид
- **Шутеры:** КиберБобер (платная, неоновый), КС2 (не поддерживается)
- **Головоломки:** 2048, Тетрис, Сапёр, Судоку, Сокобан, Пятнашки
- **Логика:** Крестики-нолики (vs компьютер), Четыре в ряд (vs компьютер), Пинг-Понг (vs компьютер)
- **Слова:** Виселица, БобрВорд (Wordle)
- **Память:** Память (Memory), Симон (со звуком)
- **Казуальные:** Камень-Ножницы-Бумага

**Скоро:** Майнкрафт Бобра, БоброКарт, Among Beavers, Бобр: RPG, Танчики, Шахматы...

## 🚀 Запуск

### Играть онлайн
Откройте https://vanya8scripts.github.io/BoberOS/ — демо развёрнуто через GitHub Pages.

### Локальный запуск
```bash
bun install
bun run dev
```
Откройте `http://localhost:3000` и пройдите установку!

### Сборка
```bash
bun run build
```
Статический экспорт появится в папке `out/`.

### 🔑 Ключ активации
```
VANYA8-BOBEROS
```

### 🚢 Деплой на GitHub Pages
Репозиторий уже настроен для GitHub Pages (`.github/workflows/deploy.yml`). При пуше в `main` сайт автоматически соберётся и опубликуется. Не забудьте в настройках репозитория (Settings → Pages → Source) выбрать **GitHub Actions**.

## 🛠️ Технологии

- **Next.js 16** (App Router, Turbopack)
- **TypeScript 5**
- **Tailwind CSS 4** + **shadcn/ui**
- **Zustand** (состояние + persist в localStorage)
- **Canvas API** (все игры)
- **Web Audio API** (звуковые эффекты)
- **lucide-react** (иконки)

## 📁 Структура

```
src/
├── app/                    # Next.js App Router
│   ├── page.tsx           # Главная (фазы boot)
│   ├── layout.tsx         # Root layout
│   └── globals.css        # Стили + тёмная тема
├── lib/
│   ├── os-types.ts        # Типы (AppId, Window, Language...)
│   ├── app-meta.ts        # Метаданные 45+ приложений
│   ├── os-store.ts        # Zustand store (persist v3)
│   ├── i18n.ts            # Переводы ru/en
│   ├── sfx.ts             # Звуковые эффекты
│   ├── wallpapers.ts      # Стили обоев
│   └── utils.ts           # cn() helper
├── components/
│   ├── os/                # Ядро: OOBE, BootScreen, Desktop, Taskbar, Windows...
│   └── apps/              # Все приложения и игры
└── public/
    ├── wallpaper1.png     # Сгенерированные обои
    └── wallpaper2.png
```

## 🎯 Как играть

1. **Пройдите установку** — выберите язык, имя, аватар, редакцию, введите ключ `VANYA8-BOBEROS`
2. **Откройте BoberCoin** (если нет ключа) → накликайте 500 монет → выведите на BoberSoft → активируйте в Настройках
3. **Скачайте Спим** в BoberStore → установите игры
4. **Для КиберБобер** (3500): накликайте в BoberCoin → выведите на Спим → купите
5. **Сохраняйте прогресс** кнопкой "Сохранить" перед выходом!
